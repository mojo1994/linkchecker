# linkchecker
a "link checker" code script write with python that scans vuln malware in link website, use with learning ethical

to use in terminal, you can follow these steps.

1 - step, update and upgrade termux or linux terminal

$: pkg update && upgrade

2 - step, instal request, colorama and python dependences

$: pkg install python && colorama && requests

3 - step, clone git hub repository
(if you dont have "git" just write)

$: pkg install git

and u can run the code

$: git clone https://github.com/mojo1994/linkchecker

$: python linkcheck.py
